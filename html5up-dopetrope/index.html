My Portfolio

Not Cyberpunk 2077

A terminal based Python project done solo. It gives the player two options for character class that determine the adventure the player will go on. A stat roller that buffs certain stats based on your character class makes for a good start to a solid RPG engine. There are also a ton of Easter eggs throughout the narrative of the game.

MusicLink

A group of 4 30+ year olds fed up with the fact that Mtv no longer plays music decided to take one back for the elder Millenials. MusicLink allows you to search for a music artist and retrieve their YouTube videos and Wikipedia artist information using their respective APIs. I was personally responsible for all the styling and was part of the API integration.

  • Search a Music Artist!

Pantry Raider

A recipe and dish suggester based on what ingredients the user had in their pantry. The dish suggestions came from our own PostgreSQL database and a recipe API. Login authentication and profile databases that can store favorite recipes for each user profile as well. I contributed to the project in the database, styling and the favorite recipes function.

  • Raid the Pantry!

Magic: the Gathering Deckbuilder

A solo project done with React and Redux. It allows you to search for the card by name or card specifics. It returns every printing of the card from an API and let's you add it to a deck.
