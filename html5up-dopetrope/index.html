Not Cyberpunk 2077

A terminal based Python cyberpunk game without all the bugs! Done solo. It gives the player two options for character class that determine the adventure the player will go on. A stat roller that buffs certain stats based on your character class makes for a good start to a solid RPG engine. There are also a ton of Easter eggs throughout the narrative of the game.

MusicLink

A group of 4 30+ year olds fed up with the fact that Mtv no longer plays music decided to take one back for the elder Millenials. MusicLink allows you to search for a music artist and retrieve their YouTube videos and Wikipedia artist information using their respective APIs. I was personally responsible for all the styling and was part of the API integration.

Pantry Raider

A full-stack project that returns recipes and dishes based on what ingredients the user has in their pantry. The dish suggestions came from our own PostgreSQL database and a recipe API. Login authentication and profile databases that can store favorite recipes for each user profile as well. I contributed to the project in the database, styling and the favorite recipes function.

Magic: the Gathering Deckbuilder

A solo project done with React and Redux. I own an online shop named World of Realms that specializes in table top gaming and producing related content. I created this to be used on our site which is launching shortly. It allows you to search for the card by name or card specifics. It returns every printing of the card from an API and let's you add it to a deck for later use. (click on the MTG Deckbuilder button)

LamentID

LamentID is a service that allows users to track their favorite brands, competitions, stocks, sports teams... the options are endless. Scanning through thousands of comments on any subreddit on reddit.com, LamentID provides data regarding the general sentiment of anything you can think of. Thinking about purchasing a new car? Allow LamentID to give you real people's opinions on your future purchases. Thinking about investing? LamentID can show you other people's thoughts on the market and stocks. Make your money count.
